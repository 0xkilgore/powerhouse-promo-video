Interactive Studio

To tweak animations, colors, timing, and props interactively:


1. Clone: git clone https://github.com/0xkilgore/powerhouse-promo-video.git

2. Install: cd powerhouse-promo-video && npm install

3. Run: npm run dev

4. Open http://localhost:3000 in your browser


Each composition can be previewed, scrubbed, and re-rendered with different props.
