How to Edit & Re-render

1. Clone: git clone https://github.com/0xkilgore/powerhouse-promo-video.git

2. Install: cd powerhouse-promo-video && npm install

3. Preview: npx remotion studio (opens browser at localhost:3000)

4. Edit text in src/Root.tsx (title cards) or individual component files in src/components/

5. Render: npx remotion render src/index.ts CompositionId out/filename.mp4

6. Deploy: cp out/*.mp4 public/videos/ && git add . && git commit -m "msg" && git push && vercel --yes --prod

Full guide: resources/remotion-text-guide.md in the main powerhouse repo
